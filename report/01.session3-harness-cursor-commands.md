## 핵심 결정·계약

- **API:** `validate_lines(grid) -> {"status": "pass"|"fail"|"incomplete", "failed_lines": list[str]}`
- **도메인:** 4×4, `0`=빈칸, `1~16`, 마법상수 `34`, 10선 `R1~R4`·`C1~C4`·`D1`·`D2`
- **SSOT (하네스):** `src/validate_lines.py`의 `MAGIC_SUM`, `LINE_IDS` — 룰에는 SSOT 선언 아직 없음
- **TDD:** RED→GREEN→REFACTOR. RED는 `tests/`만 수정. assert 완화·skip·xfail 금지
- **Git:** `origin/staging` 푸시 완료. `spec`은 로컬만. commit은 사용자 요청 시만
- **구현 상태:** `validate_lines` 본문 미구현 (`...`). 테스트 본문 없음. RED 미시작
